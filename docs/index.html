致敬原作

基于 @re-ovo 的 Rikkahub 重构

本项目是 @re-ovo 所设计与开发的开源项目 Rikkahub 在 Windows 平台的移植版本。所有产品方向、命名、视觉品牌与产品理念均归原作者所有。本桌面版仅在 Windows 平台对其重构实现，并尽可能保持与原作的功能与数据格式一致 — 你可以在 PC 端直接导入手机端备份。

功能特色

为 AI 爱好者打造

每个细节都经过实际场景打磨。

多供应商支持

OpenAI / Anthropic / Google + 任意 OpenAI 兼容端点。一个客户端聊遍全网模型。

本地模型即开即用

Ollama / LM Studio / llama.cpp Server — 端点指向 localhost:11434 即可。

MCP 与内置工具

Model Context Protocol + Streamable HTTP，Gemini Search Grounding 与 OpenAI web_search 一键切换。

世界书 · 提示词注入 · Skills

世界书条目按触发词自动注入，提示词注入按位置叠加到系统提示词，Skills 可从 GitHub 一键导入。三套机制可以混合使用，每个助手独立配置。

17 种联网搜索

Tavily、Exa、Brave、Perplexity、博查、智谱、秘塔、Firecrawl、Grok、Jina、SearXNG、自定义 JS……

完整 Markdown 渲染

代码高亮、LaTeX 公式、Mermaid 图、表格、思维链与代码块预览。

分支与重新生成

在任意消息处生成多个分支，每个分支可独立切换模型对比。

Android 备份直接导入

.zip 备份一键导入 — 对话历史、设置、附件、Skills、MCP、提示词注入、世界书全保留。

TTS 与 ASR

Windows 系统语音、OpenAI、Gemini、Qwen、Groq、MiniMax、MiMo，带一键测试。

供应商管理

所有供应商，一处管理

内置 OpenAI、Anthropic、Google、AiHubMix、硅基流动、DeepSeek、OpenRouter、Vercel AI Gateway、阿里云百炼、火山引擎、月之暗面、智谱、阶跃星辰 等 30+ 供应商模板，添加 API Key 即用。

支持自定义 Base URL、Response API、自定义请求头/请求体、Claude Prompt Caching、余额查询。每个模型还可设独立的供应商覆盖。

文本到图像

支持 gpt-image-2、DALL·E 3、Imagen、Qwen-Image、FLUX 等主流模型。可上传参考图编辑，自动保存历史参考便于复用。

扩展生态

为每个助手注入个性

MCP 服务器、提示词注入、世界书、快捷消息、Agent Skills — 五种扩展机制可以混合使用。每个助手独立选择启用哪些扩展，互不干扰。

世界书条目按触发关键词自动注入上下文；提示词注入按"系统提示词前后 / 聊天开头结尾 / 指定深度"位置叠加；Skills 直接从 GitHub 仓库一键导入。
